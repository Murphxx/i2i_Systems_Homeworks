Bu projede, Apache Kafka kullanarak Java ile basit bir veri akışı sistemi geliştirdim. Docker Compose kullanarak Kafka ve Zookeeper servislerini ayağa kaldırdım. Ardından Student adında bir Java sınıfı oluşturdum ve bu sınıftan oluşturduğum nesneyi Kafka Producer aracılığıyla serileştirip student-topic adlı bir topic’e gönderdim. Consumer tarafında, bu nesneyi Kafka’dan okuyup deserialize ettim ve konsola yazdırdım. Java nesnelerini byte dizisi (byte[]) olarak iletmek için SerializationUtils sınıfını kullandım. Bu projeyle birlikte Kafka’nın temel veri akışı mantığını uygulamalı olarak öğrenmiş oldum ve Kafka’yı Java uygulamalarıyla nasıl entegre edebileceğimi deneyimledim.
